## Install

1. Download or copy the code content from [dracula-theme.js](/dracula-theme.js)

1. In Delver X, navigate to the Extensions page, by clicking on the left bar icon :puzzle_piece:, tap Create then Install.

1. Select the dracula-theme.js file you downloaded, or paste the copied code from it

1. The new, dracula dark theme will be applied immediately.
