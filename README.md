## Install

---

1. Download [`dracula-theme.js`](/dracula-theme.js), or copy its source code.

2. Open Delver X.

3. Navigate to the Extensions page by clicking `🧩` on the left sidebar.

4. Click **Create**, then **Install**.

5. Select the downloaded `dracula-theme.js` file, or paste the copied source code.

6. The Dracula dark theme will be applied immediately.

## Color Palette

---

### Dracula Classic

| Token                       | Hex       | Usage                                 |
| --------------------------- | --------- | ------------------------------------- |
| `primary`                   | `#ff79c6` | Main primary color                    |
| `primary-dark`              | `#e66db4` | Darker shade of the primary color     |
| `primary-light`             | `#ff8ec8` | Lighter shade of the primary color    |
| `secondary`                 | `#bd93f9` | Secondary color                       |
| `dark-text-color-100`       | `#f8f8f2` | Main text color for dark mode         |
| `dark-text-color-200`       | `#ecece6` | Secondary text color for dark mode    |
| `dark-text-color-300`       | `#8be9fd` | Tertiary or disabled text color       |
| `dark-background-color-100` | `#282a36` | Base background for dark mode         |
| `dark-background-color-200` | `#343746` | Primary surface color for dark mode   |
| `dark-background-color-300` | `#424450` | Secondary surface color for dark mode |
